## 问题描述

使用Promise封装异步请求数据时，当请求失败，可重试指定次数，最好封装一个`retry(fun:Promise, times:Number)`的重试工具函数

## 方法实现

```javascript
function retry(fn, times){
    return new Promise((resolve, reject)=>{
        function run(){
            fn().then(resolve).catch(err=>{
                if(times--){
                    console.log(`还有 ${times} 次尝试`)
                    run()
                }else{
                    reject(err)
                }
            })
        }
        run()
    })
}
```
